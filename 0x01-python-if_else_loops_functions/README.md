0x01. Python - if/else, loops, functions
========================================

This project practices **control flow** in Python: conditionals (`if`/`elif`/`else`), loops (`for`, `while`), and defining functions. You write scripts that use comparisons, loops, and return values.

Tasks
-----

### 0. If/else, loops, functions (tasks 0–12)

mandatory

Each file solves a small problem using if/else, loops, or functions: e.g. check positive/negative/zero, print last digit, loop through a range, or define a function that returns a value. Run: `python3 <task_file>.py` (e.g. `python3 0-positive_or_negative.py`).
